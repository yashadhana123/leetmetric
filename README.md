# 🧮 LeetMetric

**LeetMetric** is a lightweight and responsive web application that fetches and displays **LeetCode user statistics** using their username. It features a clean UI with animated circular progress indicators for **Easy**, **Medium**, and **Hard** problems. Designed using modern HTML, CSS, and JavaScript, it provides a smooth and user-friendly experience with no backend required.
